# Contributing

Thanks for contributing to `codeitamarjr/laravel-attachments`.

## Local setup

```bash
composer install
composer test
```

## Guidelines

- Keep changes focused and small where possible.
- Add or update tests for public behavior changes.
- Update the README when installation, configuration, or API behavior changes.
- Prefer Laravel-native patterns and community conventions.

## Pull requests

- Describe the problem being solved.
- Summarize the chosen approach and any tradeoffs.
- Include test coverage for the new or changed behavior.
